# Bangladesh B2B Marketplace — Simple Overview
### What We Are Building & Who It Is For

> This document is written in plain language for non-technical readers.
> No jargon. No code. Just clear explanations of what the platform does.

---

## What Is This Platform?

Think of it as a **free online marketplace** — but only for businesses.

Instead of one person selling to another person (like Bikroy or Facebook Marketplace),
this platform connects **suppliers and manufacturers** with **wholesale buyers and businesses**.

```
  ┌─────────────────────────────────────────────────────────────────┐
  │                                                                 │
  │    SUPPLIER / MANUFACTURER          BUYER / BUSINESS           │
  │                                                                 │
  │    "আমি কাপড় বানাই।              "আমার ৫০০ গজ কাপড়           │
  │     কে কিনবে?"                     দরকার। কোথায় পাব?"          │
  │                                                                 │
  │            │                                │                  │
  │            └──────────────┬─────────────────┘                  │
  │                           │                                    │
  │                           ▼                                    │
  │              ┌─────────────────────────┐                       │
  │              │   Our B2B Marketplace   │                       │
  │              │   🏪 Free for Everyone  │                       │
  │              └─────────────────────────┘                       │
  │                                                                 │
  └─────────────────────────────────────────────────────────────────┘
```

**100% Free.** No subscription. No hidden fees. Ever.

## Admin Responsibilities

| Task | What the Admin Does |
|------|---------------------|
| **Approve products** | Every new product listing is reviewed before going live on the site |
| **Reject products** | If a product has a bad photo, misleading info, or is fake — reject it with a reason so the seller can fix it |
| **Verify sellers** | After checking a business is real, add a ✅ "Verified" badge to the profile |
| **Manage categories** | Add new product categories (e.g., "Plastics", "Paper & Packaging") and subcategories |
| **Manage users** | Suspend sellers/buyers who violate rules |
| **Edit static pages** | Update the About Us, Terms of Service, Privacy Policy pages |
| **Upload banners** | Add promotions, announcements at the top of the homepage |
| **Send emails** | Broadcast a message to all sellers or all buyers |
| **Download reports** | Export user lists, product lists, enquiry data to Excel |

# ✅ What We Are NOT Building

It is just as important to know what is **out of scope**:

| Not Building | Reason |
|--------------|--------|
| ❌ Payment processing | Buyers pay sellers offline — not through the platform |
| ❌ Delivery / logistics | Sellers handle shipping themselves |
| ❌ Subscriptions or plans | Platform is 100% free, no tiers |
| ❌ Product inventory management | We are a directory / marketplace, not a stock system |
| ❌ Auctions | All pricing is fixed range, not auction-based |
| ❌ A mobile app (Phase 1) | Website works perfectly on mobile browsers |
